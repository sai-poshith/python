# Student Marks Manager

This is a simple Python console-based mini project.

## Features
- Takes student names and marks as input
- Stores data using a dictionary
- Saves student data into a file
- Reads data from file
- Displays Pass / Fail result for each student

## Concepts Used
- Variables and input
- Loops
- Dictionaries
- File handling
- Conditional statements

This project was created as part of a 30-day Python learning plan.
